Numem es una app creada con el fin de ejercitar la memoria de el usuario con el reto de memorizar los numeros aleatorios que se muestran en pantalla cuando se inicia el juego, una vez que se termine de mostrar dichos numeros, el programa procedera a hacer visible un campo de dentrada en el cual el usuario debe ingresar en orden aquellos numeros que se mostraron anteriormente, si el usuario ingresa los numeros correctamente y en el orden que se le dio puede seguir jugando y su puntaje subira un punto.


Si el usuario llega hasta cierto puntaje y pierde, dicho puntaje se reducira a 0, pero cabe aclarar que su record permacecera hasta que pueda superarlo.


en esta ultima version se agregaron opciones para la cantidad se numeros que desea memorizar.
